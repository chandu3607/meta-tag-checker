# MetaTagsChecker - Meta Tags Preview & Generator

A comprehensive Next.js tool for previewing, editing, and generating meta tags with live previews for Google, Facebook, Twitter, LinkedIn, Slack, and WhatsApp.

## Getting Started

1. **Install dependencies:**
```bash
npm install
```

2. **Run the development server:**
```bash
npm run dev
```

3. **Open your browser:**
Navigate to [http://localhost:3000](http://localhost:3000)

## Troubleshooting

If the project doesn't run:

1. Delete `node_modules` and `.next` folders
2. Run `npm install` again
3. Run `npm run dev`
4. Check the console for any errors

If you see module not found errors, ensure all component files exist in the `components` folder.
